注意：js 发送 json 数据时，发送的是 json 字符串而不是 json 对象。比如 {'data': 'silence'} 是 json 对象，发送时会报解析 json 失败的错误。应该发送的是 "{'data': 'silence'}"，json 对象转换为 json 字符串可以使用 JSON.stringify()。

json 字符串转换为 json 对象可以使用 JSON.parse()。
